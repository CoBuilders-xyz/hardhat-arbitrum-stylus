Guides provide task-oriented instructions for common development scenarios.

## About Guides

Guides are **how-to** documents that help you accomplish specific tasks. Unlike [tutorials](../tutorials/index.md) which teach concepts step-by-step, guides assume you understand the basics and need focused instructions for a particular goal.

## Available Guides

### [Local Development](local-development.md)

Set up and optimize your local development workflow with Hardhat Arbitrum Stylus.

**For:** Developers setting up their daily workflow

### [Testing Stylus Contracts](testing-stylus.md)

Strategies and patterns for testing Stylus contracts effectively.

**For:** Developers writing tests for Stylus contracts

### [Troubleshooting](troubleshooting.md)

Solutions to common issues and error messages.

**For:** Anyone encountering problems

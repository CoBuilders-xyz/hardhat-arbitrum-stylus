# @cobuilders/hardhat-arb-node

Hardhat plugin for running a local Arbitrum node.

> This plugin is part of [@cobuilders/hardhat-arbitrum-stylus](../hardhat-arbitrum-stylus). Install the main package to use it.

## Task

```
npx hardhat arb:node
```

Starts a local Arbitrum node for development and testing.

# @cobuilders/hardhat-arb-deploy

Hardhat plugin for deploying Arbitrum Stylus contracts.

> This plugin is part of [@cobuilders/hardhat-arbitrum-stylus](../hardhat-arbitrum-stylus). Install the main package to use it.

## Task

```
npx hardhat arb:deploy
```

Deploys Solidity and Stylus contracts.

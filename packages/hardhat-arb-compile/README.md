# @cobuilders/hardhat-arb-compile

Hardhat plugin for compiling Arbitrum Stylus contracts.

> This plugin is part of [@cobuilders/hardhat-arbitrum-stylus](../hardhat-arbitrum-stylus). Install the main package to use it.

## Task

```
npx hardhat arb:compile
```

Compiles Solidity and Stylus contracts.
